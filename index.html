Hello, World!

This is an attempt at twee tech humour, a personal homepage parodically styled after the Apache2 Debian Default Page. Normally, if you saw a page that looks like this, it would mean that the Apache HTTP server installed at this site was working properly. An administrator would replace this file (located at /var/www/html/index.html) before continuing to operate their HTTP server.

This webpage's primary objective is simply to occupy the index location for this domain. Beyond the original after which this page is styled, this website uses responsive layouts and adjusts its colour palette to your device's preferred colour scheme; light or dark. As you are assumedly a human user if you are reading this, thank you for showing an interest.
